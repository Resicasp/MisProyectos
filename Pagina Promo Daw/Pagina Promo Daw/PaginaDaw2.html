Desarrollo de Aplicaciones Web


Siendo Técnico Superior en Desarrollo de Aplicaciones Web (DAW), podrás desarrollar, implantar y mantener aplicaciones web, con independencia del modelo empleado y utilizando tecnologías específicas, garantizando el acceso a los datos de forma segura y cumpliendo los criterios de accesibilidad, usabilidad y calidad exigidas por los estándares establecidos.

Aprenderás a programar en distintos lenguajes y marcos, a crear, desarrollar e implementar aplicaciones diseñadas para su funcionamiento en un entorno web; y serás capaz de gestionar servidores de aplicaciones adaptándolas para permitir el despliegue de aplicaciones web, crear aplicaciones web con acceso a bases de datos, integrar componentes multimedia en el interface de una aplicación web y Desarrollar e integrar componentes software en el entorno del servidor web.

Asignaturas del curso:

  • Bases de datos.
  • Desarrollo web en entorno cliente.
  • Desarrollo web en entorno servidor.
  • Despliegue de aplicaciones web.
  • Empresa e iniciativa emprendedora.
  • Diseño de interfaces web.
  • Entornos de desarrollo.
  • Formación en centros de trabajo.
  • Formación y orientación laboral.
  • Lenguajes de marcas y sistemas de gestión de información.
  • Programación.
  • Proyecto de desarrollo de aplicaciones web.
  • Sistemas informáticos.
